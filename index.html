가장 확실 — 넷튜터 "성적표 출력"으로 저장한 PDF 파일을 [PDF 불러오기]로 올리거나, 페이지에서 Ctrl+A → 복사 → 붙여넣기. (둘 다 인터넷 연결이 필요 없습니다.)
또는 성적표 링크 주소를 붙여넣어도 됩니다(망 상태에 따라 가끔 느릴 수 있음).

※ 반드시 ‘상세 성적표’를 사용하세요. ‘요약 성적표’에는 누적 성취도·문항별 데이터가 없어 회차 비교와 영역 분석이 되지 않습니다.
표시 방식 학부모용은 등급/단계, 내부 기록은 숫자로 전환됩니다.
AI 정밀 분석 ✅ 서버(프록시)에 API 키가 설정되어 있어 키 입력이 필요 없습니다. 바로 사진 채점을 사용하세요.
답안지 생성기 선택한 시험의 코드 박힌 답안지를 만들어 인쇄합니다.
채점 시험 선택 답안지 상단 코드와 같은 시험을 고르면 위 ‘채점 프리셋’이 자동 설정됩니다.
